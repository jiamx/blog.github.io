hive 内部表和外部表的区别

  • 建表时带有external关键字为外部表，否则为内部表
  • 内部表和外部表建表时都可以自己指定location
  • 删除表时，外部表不会删除对应的数据，只会删除元数据信息，内部表则会删除
  • 其他用法是一样的
